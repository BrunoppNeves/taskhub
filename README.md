# 🧠 TaskHub - Backend

Este é o backend do **TaskHub**, um sistema Full Stack de gerenciamento de tarefas desenvolvido em **Java (Spring Boot)** e **React**.  

---

## 🚀 Tecnologias Utilizadas

- **Java 17+**
- **Spring Boot 3**
- **Spring Data JPA**
- **Spring Web**
- **MySQL**
- **Docker**
- **Lombok**
- **Validation (Jakarta)**
- **Springdoc OpenAPI (Swagger UI)**
